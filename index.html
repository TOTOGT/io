yoyo — an index that goes around, comes around

This repo holds proof templates, drafts, and loose material that orbits the main Principia Orthogona / dm³ series without belonging to a single volume — what goes around, comes around, like a yoyo on its string.

oi oi oi → reversed → io io io
first-wave punk chant, run backwards, becomes the imaginary unit and the loop

The name is a small joke that is also the point: io is not the opposite of real. Run a thing backwards and what looked like noise turns into $i$ and a circle — exactly the move Book 6 opens with. See the AXLE repo for the formal side and geometry/book6 for the prose framing.
